PROJECT TITLE: Pikachu's Strawberry Run
PURPOSE OF PROJECT: To help Pikachu collect all the strawberries you find.
VERSION or DATE: 2/12/2021
HOW TO START THIS PROJECT: 
AUTHORS: Jean Zachary Reyes
USER INSTRUCTIONS: Use the up arrow key to move up and down arrow key to move down.
collect all strawberries you see to get a score. 1 strawberry = 1 point. You have 5 health
lose 1 hp if you hit meowth, the game will stop if your health is 0.
